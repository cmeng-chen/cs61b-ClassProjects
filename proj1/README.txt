cs61b Project1 - A Small Database System

Author: Paul Hilfinger & Chen Meng

Description: implement a miniature relational database management system (DBMS) that stores tables of data, where a table consists of some number of labeled columns of information.

Main functions:
- create, store, print and load tables; insert values into tables
- select clauses used to create tables from multiple other tables
- condition clauses used in select clauses to restrict the conditions for selections
